What Tare does

Chrome's Memory Saver evicts tabs by recency, which often drops your banking tab while keeping a Reddit feed. Tare replaces that with typed lifecycles.

!
Reusable — critical sessions. Protected from auto-eviction. Banking, email, calendar, AI chats, admin panels.

Linear — use-once references. Auto-discharged after idle. Searches, Wikipedia, Stack Overflow, docs.

Affine — done with these. Evicted first under memory pressure. Social feeds, news homepages, old searches.

How to use

Auto-typing: Tabs are classified by URL as you open them.

Manual override: Click any type button on a tab, or right-click a page → Tare · set tab type.

Bulk actions: "drop affine" discards all affine tabs now. "discharge idle" discards linear tabs past idle. 30-second undo window.

Automatic: Every tick interval (default 2 min), linear tabs past idle are discharged. When system RAM exceeds your threshold (default 85%), affine tabs auto-evict with notification.

Limits

Chrome's own Memory Saver still runs in parallel. Protection is advisory — if the OS is truly out of memory, tabs can still be killed. No cross-device sync yet.
